# 示例合集

## 目录下全部自动进导航

`docs/examples/` 里只要放 **Markdown 页面**（含任意层子目录里的 `index.md` 等），**不必**在 `.pages` 里逐个写条目。

本目录的 `examples/.pages` 只固定把 [概述](index.md) 放在最前，其余全部交给 **`...`** 自动收录（含 `2019-10/某文章/index.md` 这类结构）。

拷贝新文章、新月份文件夹到 `examples/` 下后，**保存并构建即可**，无需再跑脚本、无需按月维护导航。

## 图片与资源

图片放在对应文章目录下，文中用相对路径引用即可。
